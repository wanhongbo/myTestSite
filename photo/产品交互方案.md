关联文档：`doc/方案 .md`

---

## 1. 产品交互目标

- 用户在 30 秒内理解核心卖点：**本地处理，不上传**
- 用户在首次会话内完成一次“扫描 -> 打码 -> 保存”
- 免费用户可以完整体验基础价值，关键场景自然触发会员引导
- 权限拒绝、支付失败、扫描中断等异常路径可恢复

## 3. 关键流程设计

## 3.1 首次使用流程（FTUE）

1. 启动页（1.5-2s）
2. 新手引导 3 屏（可跳过）
3. 权限说明页（先解释用途，再触发系统权限）
4. 进入首页并突出“开始扫描”
5. 扫描完成进入结果页
6. 完成首张照片打码并保存

完成判定：用户首次会话完成步骤 1-6 中至少 5 步。

## 3.2 核心处理流程（主流程）

1. 首页点击 `开始扫描`
2. 扫描页展示进度、预计时间、暂停/取消
3. 完成后自动进入结果页
4. 结果页按类别筛选
5. 进入打码编辑页调整遮罩
6. 保存到私密相册（或导出副本）
7. 返回结果页显示“已处理”

## 3.3 会员转化流程（自然触发）

触发场景：

- 结果页点击“批量处理”
- 导出去水印
- 私密相册超出免费容量

流程：

1. 打开会员页（权益对比 + 套餐）
2. 选择月/年/终身
3. 系统支付
4. 支付成功后回来源页并刷新权益

失败处理：

- 支付失败：保留页面状态 + 重试按钮
- 成功未生效：提供“恢复购买”一键修复

## 4. 页面交互规格（逐页）

## 4.1 启动页

- 元素：Logo、品牌名、隐私承诺短句
- 行为：自动跳转；非首次直接进首页
- 文案示例：`On-device. No upload.`

## 4.2 新手引导页

- 每屏：主图 + 标题 + 描述 + 下一步
- 第三屏 CTA：`开始保护照片`
- 支持手势滑动与“跳过”

## 4.3 权限引导页

- 先显示业务解释：为何要相册权限
- 点击“继续”后再触发系统权限弹窗
- 若拒绝：显示“去设置开启” + 提供降级路径（仅浏览，不扫描）

## 4.4 首页

- 顶部：隐私承诺条（可关闭）
- 主 CTA：`开始扫描`
- 次入口：私密相册 / 打码工具 / 分类
- 最近记录：最近处理的图片或任务摘要

## 4.5 扫描页

- 实时进度：已扫描数/总数/预计剩余时间
- 控件：暂停、后台继续、取消
- 中断恢复：再次进入自动恢复未完成任务

## 4.6 结果页

- 顶部摘要：命中总数 + 分类别数量
- 筛选器：全部、证件、银行卡、手机号、车牌
- 列表项：缩略图、风险标签、处理状态、进入编辑
- 批量处理按钮：免费触发会员页；会员直接执行

## 4.7 打码编辑页

- 默认加载 AI 识别框
- 手势：拖拽、缩放、删除遮罩
- 样式：马赛克 / 模糊 / 实色遮挡
- 工具栏：撤销、重做、重置、保存
- 保存去向：私密相册 / 导出副本 / 覆盖原图（二次确认）

## 4.8 私密相册页

- 解锁方式：密码 + 生物识别（会员）
- 视图：网格、时间分组
- 操作：多选删除、移动分组、导出
- 风险操作：二次确认 + 可撤销提示

## 4.9 会员购买页

- 展示：免费版 vs 会员版能力差异
- 套餐：月 / 年（主推）/ 终身
- 必备文案：自动续订说明、取消路径、服务条款
- 操作：购买、恢复购买

## 4.10 我的页 / 设置页

- 当前计划状态（Free/Premium）
- 入口：恢复购买、语言、隐私政策、服务条款、反馈
- 支持切换语言并即时生效（需重绘文案）

## 5. 状态机定义（开发必须一致）

## 5.1 权限状态

- `unknown`
- `granted`
- `denied_once`
- `denied_forever`

规则：`denied_forever` 仅引导去系统设置，不重复弹系统授权。

## 5.2 扫描任务状态

- `idle -> preparing -> scanning -> completed`
- 异常分支：`scanning -> paused/failed/cancelled`
- 恢复：`paused -> scanning`

## 5.3 会员状态

- `free`
- `premium_active`
- `premium_expired`
- `verifying`（支付后短暂校验）

---

## 6. 异常与边界交互

- 无照片：空状态 + `去拍照` / `导入照片`
- 模型加载失败：提示重试 + 上报日志
- 空间不足：提示清理并提供仅处理不保存选项
- 弱网：核心本地能力可用，支付能力降级提示
- 多语言长文案：按钮允许换行，禁止裁切省略关键动作词

---

## 7. 多语言交互规范

- 所有 UI 文案走 i18n 资源文件，不允许硬编码
- 同一术语保持一致：
  - Private Vault
  - On-device Processing
  - Sensitive Scan
- 数字/货币按 locale 格式化显示（例如 `$2.99/mo`）
- 付费链路文案必须人工校对（避免机翻影响转化）

## 9. 交互验收清单（QA/联调用）

- 可在 3 步内从首页进入扫描
- 权限拒绝后存在明确恢复路径
- 底部 Tab 点击区域 >= 44x44
- 筛选切换无明显卡顿或错乱
- 打码编辑支持撤销/重做/重置
- 购买成功后权益在 3 秒内刷新
- 恢复购买可独立修复权益状态
- 多语言下无主要文案截断
- 法务页与商店声明一致
